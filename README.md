
## Installation

Step 0:

##### `npm install`

Step 1:

##### `npm install axios`

Step 2:

Terminal 1
##### `npx json-server --port=3001 --watch db.json`

Step 3:

Terminal 2
##### `npm start`

Step 4:

##### `http://localhost:3000/`

## To run tests

Step 5:

Terminal 3
##### `npm run cypress:open`
